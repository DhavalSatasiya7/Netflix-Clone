# Netflix-Clone
Netflix Clone is a responsive front-end web application inspired by the official Netflix landing page. The project is developed using HTML5 and CSS3 to recreate the modern and attractive user interface of Netflix.

🎬 Netflix Clone
📌 Project Overview

Netflix Clone is a responsive web application designed to replicate the user interface of the Netflix homepage. The project showcases modern web design techniques and front-end development skills using HTML5 and CSS3.

The website provides a visually appealing streaming platform layout with sections for featured content, trending movies, subscription options, service features, frequently asked questions, and footer navigation links.
